  17. def parseSelected(jsonString: String): Either[List[ComboItem], Selected]

    Parse JSON data from select2.

    Parse JSON data from select2.

    This method will parse JSON returned by select2 to Scala object.

    If the combo box is allow multiple value, then it will return a Scala's Left[List[ComboItem]] which contains the item that user selected.

    If the combo box is single value, then it will return Some[(selectedItem, isNewItem)] if there is value been selected or it will return None if user unselected current value.

    jsonString

    The JSON data returned by select2.

    returns

    The corresponding Scala object.
